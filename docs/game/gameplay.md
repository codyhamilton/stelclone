# Gameplay Systems

## Time Control

The game runs in real-time but allows players to control the rate of time progression. This system enables both detailed micromanagement and rapid progression through quiet periods.

### Time Control Options
- **Paused**: Game time is completely stopped
- **Slow**: 1 game day = 10 seconds real-time
  - Best for detailed combat or complex operations
  - Allows precise control during critical moments
- **Normal**: 1 game day = 5 seconds real-time
  - Default speed for general gameplay
  - Balanced between control and progression
- **Fast**: 1 game day = 2 seconds real-time
  - Useful for speeding through routine operations
  - Good for long-distance travel
- **Fastest**: 1 game day = 0.5 seconds real-time
  - Maximum speed for rapid progression
  - Best for skipping through quiet periods

### Time Control Features
- Time control can be adjusted at any moment
- Automatic pause on important events (combat, colony alerts)
- Visual indicators for current time speed
- Option to set automatic speed changes based on events

## Game Screens

### Main Menu
- **New Game**: Start a new game with customizable settings
- **Load Game**: Access saved games
- **Settings**: Configure game options, controls, and graphics
- **Credits**: View game credits and acknowledgments
- **Exit**: Return to desktop

### Sector Map
- **Overview**: Zoomable 2D map of the game sector
- **Display Elements**:
  - Asteroids and celestial bodies
  - Fleet positions and movements
  - Colony locations
  - Points of interest
  - Zones of control
  - Sensor ranges
  - Resource deposits
  - Trade routes
- **Controls**:
  - Zoom in/out
  - Pan view
  - Select objects
  - Toggle display layers
  - Set waypoints
  - Issue fleet orders

### Building Info Screen
- **Basic Information**:
  - Building type and name
  - Current status
  - Power usage/output
  - Staff requirements
  - Maintenance costs
- **Detailed Stats**:
  - Production rates
  - Efficiency metrics
  - Resource consumption
  - Staff satisfaction
- **Management Options**:
  - Adjust production levels
  - Modify staff assignments
  - Set maintenance priorities
  - Configure automation settings

### Blueprints Screen
